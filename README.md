# ScamDNA 🛡️

**AI-Powered Scam Psychology & Pattern Analyzer**

ScamDNA helps users identify and understand suspicious WhatsApp, SMS, Email, and social media messages by analyzing the psychological manipulation tactics used by scammers. Instead of simply labeling a message as a scam, ScamDNA explains *why* it is dangerous.

> AI analysis — use as guidance only.

---

## 🌟 Features

### Scam Analysis

* Detects suspicious messages
* Identifies scam categories
* Generates a risk score
* Provides actionable recommendations

### Psychological Analysis

ScamDNA analyzes:

* Fear
* Urgency
* Authority
* Greed
* Curiosity
* Trust Manipulation
* FOMO (Fear of Missing Out)

### Explainable AI

* Trigger sentence highlighting
* Manipulation flow detection
* Human-readable explanations
* Transparent risk assessment

### Multilingual Support

Supports major Indian languages including:

* English
* Hindi
* Gujarati
* Marathi
* Bengali
* Tamil
* Telugu
* Kannada
* Malayalam
* Punjabi
* Odia
* Assamese

### Privacy First

ScamDNA never stores original user messages.

Stored data:

* Message Hash (SHA-256)
* Risk Score
* Scam Type
* Analysis Timestamp

## 🔍 How It Works

1. User pastes a suspicious message.
2. Gemini AI analyzes the content.
3. ScamDNA extracts:

   * Scam Type
   * Psychological Factors
   * Trigger Sentences
   * Manipulation Flow
4. Risk Score is calculated using deterministic rules.
5. A ScamDNA Report is generated.

---

## 🎯 Problem Statement

Millions of users receive scam messages every day through SMS, WhatsApp, email, and social media. Traditional scam detectors often provide only a warning without explaining the reason.

ScamDNA bridges this gap by exposing the psychological tactics used by scammers, helping users learn and protect themselves from future attacks.

---

## 💡 Innovation

Unlike traditional scam detection systems, ScamDNA focuses on **scam psychology analysis**.

Instead of saying:

> "This message may be a scam."

ScamDNA explains:

> "This message uses Authority → Fear → Urgency tactics to pressure the victim."

This improves awareness, digital literacy, and decision-making.
